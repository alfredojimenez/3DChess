Detailed Description

This class will control the gameflow of the program. This class implements the maintaining running loop for this application, and has basic functions that control chess gameflow, such as execute move, cancel game, and setupNewGame

synchronized void controller.GameLoop.executeMove ( ChessMove  move)

Executes a chess move, sets up an animation, and switches the currentPlayer. The ChessMove is also added the total move list, so it can be undone later if necessary. This move also checks to see if there is any moves left for the other to make, otherwise the game is ended If their is no game running, this method does nothing.

Parameters
moveChessMove to be executed, must be a validMove
void controller.GameLoop.run ( )

The method is what runs when start() is called on a thread. The method will continue running until running is set to "false", ie the game is over. When running is set to false, the winner will be displayed and new game will be set up.
