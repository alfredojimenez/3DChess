Detailed Description

Handles any Keystrokes or Mouse-clicks that occur within the chess frame. The methods interprets the clicks, and passes along commands to the GameLoop or Renderer

This method interprets a click within the chess window. It uses the camera, to find where that click translates into 3D space (ie where it intersects the board). This method handles the cases where there is already of selected a piece, and what to do with that piece
