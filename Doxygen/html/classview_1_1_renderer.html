Detailed Description

The Render class implements the functionality of a GLEventListener which defines the methods that are called by the underlying GLCanvas. Upon create of the GLCanvas, the GLCanvas calls init(), reshape(), then display(), which are each defined here. Any calls to repaint window, will eventually lead to a display() call defined here
